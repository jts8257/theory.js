### 실행 컨텍스트와 클로저
<p> JS가 실행될 때 생성되는 하나의 실행 단위인 실행 컨텍스트, 변수의 유효 범위, 클로저의 개념을 설명함. 이들을 정확히 이해하지 못하면 다른 사람이 작성한 JS코드를 이해하기 어렵고, 활용하기 불편할 뿐만 아니라 디버깅도 굉장히 어려울 것임.</p>

* 활성 객체(Activation Object)와 변수 객체(variable Object)

* 클로저 (Closure)
